iOS uygulamanızı geliştirirken balon şeklinde etiketler oluşturmak istediğiniz durumlar zaman zaman olabilir.

Benim olmuştu. Buldum, araştırdım, öğrendim ve sıra yazılım dünyasıyla paylaşmaya geldi 😊
